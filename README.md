# my-shell

A simple shell built in C 

## Features
- Command execution via PATH search
- Input/output redirection (`<` and `>`)
- Piping  (`|`)
- Background processing (`&`)
- Builtins: `cd`, `exit`, `jobs`
- Environment variable expansion (`$VAR`)
- Tilde expansion (`~`)

## Compile
```bash
make clean
make
```

## Run
```bash
./bin/shell
```
